Part 3 : 資料視覺化

台股大盤對數報酬、RV&IV趨勢圖

歡迎來到我們的RV預測Project第三集!!¶

狀況複習¶

在Part 1我們整理好了研究所必須的原始資料，包括台股大盤5秒日內資料、選擇權收盤價日資料以無風險利率日資料，並將其存放在我們的RV_predict_project database中。

在Part 2我們將原始資料做些轉換整理，我們用台股大盤5秒日內資料計算出該日的實質波動度(Realized Volatility, RV)，也利用台股收盤價、選擇權權利金、權利到期日、履約價跟無風險利率，配合py_vollib_vectorized套件計算每日的隱含波動度(Implied Volatility, IV)，並與台灣期貨交易所得IV計算器做驗證。

在這個Part 3的部分，我們將資料進行視覺化，這也是一種資料轉換的藝術，將其從數字轉換成圖像，並藉此發現些資料的內涵(Insight)，並給我們後續模型輸入時一些直覺思考。

要做的視覺化為

  1. 每日大盤指數log return做分佈圖
  2. 每日RV趨勢圖
  3. 找出RV最劇烈波動的時間點，觀察前後選擇權的IV樣態
  4. 利用交易量做加權平均去得到代表該日IV的數值，並觀察其與RV的關係

把Part 2寫好的function放在一起使用，直接取得該日的選擇權IV¶

def get_IV(date):
    opt_df = get_close_optData(date,conn) # 取的該日的option收盤資料
    closed_price = get_daily_close_TAIEX(date, conn) # 取的該日的收盤資料
    due_date = get_due_date(date)[1] # 計算該日 最快到期的選擇權的剩餘天數
    IR = get_Risk_free_IR(date, conn) # 取得該日的無風險利率
    
    # 計算IV
    f_opt_df = get_vector_IV(opt_df, closed_price, due_date, IR)
    # 並將nan的資料去除(那些option低於內含價值，無法計算IV)
    f_opt_df.dropna(inplace = True)
    
    # 計算價性指標
    f_opt_df['moneyness'] = f_opt_df['履約價'] / closed_price
    
    # 計算IV用交易量做加權平均
    f_opt_df['成交量比率'] = f_opt_df['成交量'] /  f_opt_df['成交量'].sum()
    f_opt_df['IV加權'] = f_opt_df['IV'] * f_opt_df['成交量比率']
    return f_opt_df
